# Download Sources of Kamailio v6.1.x from GIT Repository

Instructions to download sources from GIT repository, specific for
stable branch 6.1 (release series v6.1.x).

## Kamailio v6.1.x

Direct download of sources from GIT for latest Kamailio v6.1.x.

``` shell
    git clone --depth 1 --no-single-branch https://github.com/kamailio/kamailio kamailio
    cd kamailio
    git checkout -b 6.1 origin/6.1
```

*Note: if your git client version does not support **â€“no-single-branch**
command line parameter, then just remove it.*

Or, for newer git versions, with direct branch 6.1 checkout:

``` shell
    git clone --depth 1 --branch 6.1 https://github.com/kamailio/kamailio kamailio
    cd kamailio
```

*Note: if you want to have the full commit history, then remove
**--depth 1** command line parameter.*

If you want the minimal clone, only the latest snapshot of branch 6.1, use:

``` shell
git clone --depth 1 --branch 6.1 --single-branch https://github.com/kamailio/kamailio kamailio
```
