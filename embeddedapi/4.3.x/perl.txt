====== Perl API for Kamailio 4.3.x Version ======

Module **app_perl** allows execution of Perl scripts from SIP router configuration file, providing access to SIP message which is processed at that time. Exported functions by **app_perl** are available in the Perl script as package Kamailio.

References:

  * **app_perl** module: [[http://kamailio.org/docs/modules/4.3.x/modules/app_perl.html|README]]
  * **Perl**: http://www.perl.org

Perl API documentation is part of module README.
